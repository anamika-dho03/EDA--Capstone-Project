# EDA Capstone Project - Global Terrorism Data

## Overview
This project analyzes global terrorism data to identify trends, patterns, and insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Increase in attacks over years
- Most affected regions and countries
- Common attack types and targets
- Casualties analysis

## Conclusion
This analysis helps understand patterns in terrorism and supports better decision-making.
